Customer mPIN reset at banking point

Tap on CUSTOMER ACCOUNT MANAGEMENT.

Enter customer account linked Mobile Number and Tap on Proceed.

Scan Customer’s Finger on Biometric Device.

Confirmation message appears on screen after verification.

Customer changing mPIN using Smart Phone

Tap on My Airtel App.

Select PAYMENTS BANK and Tap on VIEW PROFILE.

Tap on Change mPIN.

Enter current mPIN.

Enter desired new mPIN.

Re-enter desired new mPIN.

mPIN changed successfully.

Customer changing mPIN using USSD Code - Feature Phone

Dial *400#.

Select My Account by pressing 2.

Press Send button.

Select Change mPIN by pressing 3.

Press Send button.

Enter Current mPIN and press Send button.

Enter Desired New mPIN and press Send button.

Re-enter Desired New mPIN press Send button.

mPIN changed successfully.

Transaction can't be carried out through default mPIN . Please change default mPIN.
